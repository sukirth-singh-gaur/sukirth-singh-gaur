# 🚀 Featured Projects

### 📝 [BloggIt](https://project-bloggit.vercel.app/) — AI-Powered Blogging Platform
> React.js · Node.js · MongoDB · Redis · Python · FastAPI · Docker · Nginx · AWS EC2

Production-deployed blogging platform with a full microservices architecture. Integrates Gemini AI for async blog summarization via Redis job queues, JWT auth with role-based access, and Dockerized services behind Nginx with HTTPS on AWS EC2.

### ⚙️ [WorkQueue](https://github.com/sukirth-singh-gaur/WorkQueue) — Background Job Processing System
> Redis · Node.js · Express · REST API

Redis-backed job queue with decoupled producers, concurrent workers, exponential backoff retry logic, and HTTP metrics endpoints for real-time observability of queue health and job success/failure rates.

### 🎵 [DispLyrics](https://github.com/sukirth-singh-gaur/DispLyrics) — Real-Time Spotify Lyrics
> Chrome Extension · JavaScript · Python · Flask · Selenium · Docker

Chrome extension + Flask backend that detects the currently playing Spotify track and injects lyrics directly into the page in real time. Containerized backend with health monitoring via extension popup.
